# PHP 

This creates a Docker container running PHP on Ubuntu, primarily for the
purposes of running unit tests with Drone.IO and various PHP extensions.

PHP 5.6 with Composer

## PHP Extensions
- php5-curl 
- php5-mcrypt 
- php5-json 
- php5-intl
- php5-imap 
- php5-xdebug 
- php5-redis 
- php5-mysql 
- php5-mongo 
- php5-sqlite
